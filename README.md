# Liquid Glass VS Code Icon

A custom, modern app icon for Visual Studio Code featuring a sleek liquid glass design aesthetic.

## Overview

This project provides an unofficial replacement icon for Visual Studio Code. The design combines contemporary glassmorphism principles with the iconic VS Code blue color palette, creating a professional alternative to the default icon.

## Included Formats

- **PNG** - High-resolution raster format suitable for all platforms
- **ICO** - Windows icon format
- **ICNS** - macOS icon format

## Installation

### macOS

1. Locate the VS Code application in your Applications folder
2. Right-click on VS Code and select "Get Info"
3. Drag the PNG icon file into the image well at the top-left of the Info window
4. Close the Info window
5. Restart VS Code to see the new icon
6. (Optional) Empty the Trash to persist the change

### Windows

1. Right-click on your VS Code shortcut and select "Properties"
2. Click the "Change Icon..." button
3. Click "Browse..." and navigate to the PNG icon file
4. Select the icon and click "Open"
5. Click "OK" to apply and close the Properties dialog
6. Restart VS Code

**Note:** For Windows, you may need to convert the PNG to an ICO format using an online converter or image editor for best results with the Properties dialog.
